Number Bases

When counting, we use the digits 0 to 9, and once those are used up, we add a decimal place and start back up at 0. But what if we only used 4 digits, 0 to 3? Or what if we added a digit, A? By doing so, we would be changing the base of the number system.

The concept of a base with numbers is how many digits are used to count. Most of the numbers we see are in base 10, which uses the digits 0 to 9. A few other common bases are binary (base 2) which uses 0 and 1, and hexadecimal (base 16) which uses 0 to 9 and A to F.

Bases are represented with a subscript after the number representing the base. For example, 102 or 12349. Any numbers without this subscript are assumed to be base 10.

Scientific Notation is very useful with number bases. As a reminder, scientific notation of a number like 354 would be 3×102+5×101+4.

The scientific notation of a number changes when the base changes. The scientific notation of 3546 would be 3×62+5×61+4.

An important thing to note, each digit in a number has to be less than its base. 5 can't be a digit in base 4, and 7 can't be a digit in base 7.

Now on to adding numbers of a different base. Essentially, this is the same as adding in base 10. Let's do an example in hexadecimal.

 45
+3F
---
 84

The 5 and D add to 12. (E, F, 10, 11, 12) The units digit is 2. Then we add 4 + 3 + 1 = 8. Our tens digit is 8. Our answer is 0x82.
NOTE: Hexadecimal numbers start with 0x and then the number, rather than the 16 subscript.

We can start with converting numbers from any base to base 10. This is very simple, just write it in scientific notation, and simplify.

Let's say we want to convert 10102 to base 10.

  1. First, we write it in scientific notation: 1×23+0×22+1×21+0.
  2. Then, we simplify: 8 + 2 = 10.

We're done! It's as simple as that! Converting to another base will be a bit more difficult, though. There are a few steps to go through:

  1. Find the highest power of the base that is less than the number.
  2. Divide the number by the base to this power. Record the quotient as the digit, and record the remainder for the next step.
  3. Take the remainder and divide it by the next (lower) power of the base. The quotient is the next digit and keep the remainder for the next step.
  4. Repeat the last step until you divide by the base. The remainder is the last digit and you're done.

Don't worry if this makes no sense. An example will make this a lot clearer! Let's convert 144 to base 5.

  1. The highest power of 5 that's less than 144 is 125 (53)
  2. 144 / 125 = 1, remainder 19. The first digit in our base 5 conversion is 1. We'll use 19 in our next step.
  3. 19 / 25 = 0, remainder 19. 19 is less than the next power of 5, so the next digit is 0, and we continue to the next power.
  4. 19 / 5 = 3, remainder 4. The next digit is 3, and we'll use 4 in the last step.
  5. Now we have divided by the base, 5. The remainder, 4, is the last digit.
  6. Our completed conversion is 10345

Changing bases is very tedious work, especially with large numbers. There's a lot of times where you want to change a number's base fast. That's why we've created a base converter. It can change any number from one base to another. Simply enter the number in one base (make sure each of the digits are valid) and enter it's base in the start base spot. Then, enter the base you want to convert it to in the end base spot. It's a helpful tool for large numbers and quick conversions!
